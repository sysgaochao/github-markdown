### 3.文本块
语法1：在连续几行的文本开头加入1个Tab或者4个空格。

    那边坐着一条可爱的小狗，它在快乐地摇尾巴...
    那边坐着一条可爱的小狗，它在快乐地摇尾巴...
    那边坐着一条可爱的小狗，它在快乐地摇尾巴...

语法2：使用一对各三个的反引号：

```
那边坐着一条可爱的小狗，它在快乐地摇尾巴...
那边坐着一条可爱的小狗，它在快乐地摇尾巴...
那边坐着一条可爱的小狗，它在快乐地摇尾巴...
```

### 4.文字高亮(使用一对反引号)
语法：
```
`可爱的` `摇尾巴`
```

### 5.换行
①.直接回车 ×  
②.上一行文本后面补两个空格 √  
③.两行文本直接加一个空行 √

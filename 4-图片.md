1、基本格式如下：
```
![alt](URL title)
```

举例子：  
```
![baidu](http://www.baidu.com/img/bdlogo.gif "百度logo")
```


实际效果：  
![baidu](http://www.baidu.com/img/bdlogo.gif "百度logo")

2、alt和title是可以省略的：

```
alt和title即对应HTML中的alt和title属性（都可省略）：
- alt表示图片显示失败时的替换文本
- title表示鼠标悬停在图片时的显示文本（注意这里要加引号）

![](http://www.baidu.com/img/bdlogo.gif)
```

实际效果：  
![](http://www.baidu.com/img/bdlogo.gif)
